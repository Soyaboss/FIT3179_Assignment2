Public Transport Insights in Kuala Lumpur

This dashboard brings together three views of Kuala Lumpur’s rail network: fare relationships between stations, a station map for spatial context, and daily ridership trends with interactive filtering.

How do fares vary between origin–destination station pairs?

Explore the fare matrix across the various stations. Use the slider below the map to filter by price or to find routes with higher or lower fares.

Hover to see the exact Origin → Destination pair and fare value.

Where are the LRT stations located?

The map shows station locations and (if the spec includes it) line filters. Use it to understand spatial coverage and proximity between lines.

How has daily ridership changed over time?

Interact with the date slider/brush or mode filter (if defined within the spec) to zoom into specific periods and compare LRT vs MRT lines.

Built with Vega-Lite. Specs loaded from local JSON files.
